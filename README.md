# Homebrew Tap for Traylinx

This is the official [Homebrew](https://brew.sh/) tap for [Traylinx CLI](https://github.com/traylinx/traylinx-cli).

## Installation

```bash
brew tap traylinx/traylinx
brew install traylinx
```

## Update

```bash
brew upgrade traylinx
```

## Uninstall

```bash
brew uninstall traylinx
brew untap traylinx/traylinx
```

## About

The formula is automatically updated when new releases are published to PyPI.
